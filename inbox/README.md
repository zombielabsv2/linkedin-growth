# Inbox — Claude App → Claude Code Sync

## How to use (from your phone)

1. Brainstorm in a Claude app project (Org Metabolism, Career Volume, etc.)
2. When you have ideas worth keeping, tell Claude:
   > "Format my ideas as a structured note for inbox sync"
3. Copy the output
4. Open GitHub mobile app → zombielabsv2/linkedin-growth → inbox/
5. Tap "+" to create a new file
6. Name it: `YYYY-MM-DD-topic.md` (e.g., `2026-04-02-life-after-ai-chapter-idea.md`)
7. Paste and commit

Next time you open Claude Code, say "process inbox" and everything
will be integrated into the right project files.

## Naming convention
- `YYYY-MM-DD-book-name-topic.md`
- Examples:
  - `2026-04-02-org-metabolism-new-case-study.md`
  - `2026-04-03-career-volume-interview-idea.md`
  - `2026-04-05-life-after-ai-governance-angle.md`
  - `2026-04-05-general-new-series-idea.md`
